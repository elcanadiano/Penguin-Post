Example Penguin Post page

This is an example of Penguin Post in action. Please contact me by email if you have any questions. Below is what you will need to implement in your HTML page. Please note that we use the jQuery library along with this. I added the <script> tag for jQuery 1.8.0.

I've also provided both the DEV and the minfied versions of all my CSS and JavaScript files. I recommend minifying them before using them on a live site.

You will need PHP 5.x with PHP mail() enabled along with jQuery 1.4+ in order to run Penguin Post in your website. Thank you for using Penguin Post, and as always, let me know if there are any issues. Please look into postcaptcha.php and set your questions first. You will also have to look at penguinpost.php to set your email settings.
